## üì¶ Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/shivamksharma/DevDonations.git
   cd DevDonations
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser at `http://localhost:3000`.
